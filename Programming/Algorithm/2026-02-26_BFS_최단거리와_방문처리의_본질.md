## 1) 오늘 푼 문제 유형

### ✔ 미로 탐색

-   격자 이동 (상/하/좌/우)\
-   모든 이동 비용 = 1\
-   목표: 시작점 → 도착점까지 최단 거리

### ✔ 숨바꼭질

-   상태: 정수 위치 (0 \~ 100000)\
-   이동:
    -   x - 1\
    -   x + 1\
    -   2 \* x\
-   모든 이동 비용 = 1\
-   목표: 최소 시간

👉 공통점: **모든 간선 비용이 동일(=1)** → 최단거리 = 최소 간선 개수 →
BFS

## 2) 왜 BFS는 비교 없이 최적해를 보장하는가?

### 핵심 원리

1.  BFS는 큐(FIFO)를 사용한다.
2.  거리(레벨) 순서대로 확장한다.
3.  간선 비용이 모두 동일하다.

탐색 순서:

    거리 0
    → 거리 1
    → 거리 2
    → 거리 3
    ...

어떤 노드를 처음 방문한 순간의 거리가 그 노드까지의 최소 거리이다.

### 예시로 보면 다음과 같다 :
```text
S(0)
 ├─ A(1)
 │   └─ C(2)
 │       └─ X(3)   ← 짧은 경로
 └─ B(1)
     └─ D(2)
         └─ E(3)
             └─ F(4)
                 └─ X(5)   ← 긴 경로
```
같은 목표 X에 대해서
- 거리 3 경로
- 거리 5 경로\
가 동시에 존재하는 상황에서 실제 큐의 변화 과정을 살펴보자.
```scss
Step 0
  Q = [S]

Step 1 (S 처리)
  Q = [A(1), B(1)]

Step 2 (A 처리)
  Q = [B(1), C(2)]

Step 3 (B 처리)
  Q = [C(2), D(2)]

Step 4 (C 처리)
  Q = [D(2), X(3)]
여기서 X가 거리 3으로 생성된다.

Step 5 (D 처리)
Q = [X(3), E(3)]
여기서 큐의 맨 앞은 X(3)
즉, E(3)은 더이상 처리되지 않는다. 왜냐하면 X(3)이 이미 도착점에 최적의 해로 왔기 때문이다.
```

## 4️) 방문처리의 본질

방문처리는 단순 최적화가 아니라 **BFS가 최단거리 알고리즘이 되게 만드는 핵심 조건이다.**

### 역할

1.  중복 탐색 방지
2.  무한 루프 방지
3.  최단거리 보장

예:

``` python
if dist[nx] == -1:
```

---

### (1) 상태 공간을 "트리"로 만든다
 그래프에는 사이클이 있다.\
하지만 방문처리를 하면 그래프가 이렇게 변하게 된다.
> 그래프 => BFS 탐색 트리

즉,
- 각 노드는 단 한 번만 확장된다.
- 부모가 한 번만 정해진다.
- 탐색 구조가 트리처럼 변한다.

---

### (2) 방문처리가 왜 "최단거리 보장"과 연결되는 거지?
BFS의 가장 중요한 성질은 다음과 같다 :
> 어떤 노드를 "처음 방문한 순간"의 거리가 최단거리

근데 방문처리를 만약 하지 않으면?\
같은 노드가
- 더 긴 경로
- 더 짧은 경로\

모두로 계속 큐에 들어가게 된다.

그렇게되면 우리는, 큐에 들어간 것들이 **최단거리인지 아닌지 계속 비교해야만 한다.**
즉, BFS의 "비교 없이 최적 보장"이 깨지게 되는 것이다.

### 방문처리가 없게 되면?
- 같은 노드가 여러 번 확장
- 더 긴 겨오가 나중에 나와도 계속 탐색
- 시간복잡도 폭발
- 구조적 최적성이 붕괴

### (3) 방문처리는 어떤식으로 하는가?

#### A. 가장 기본형: `visited` 배열 사용
- 단순 탐색
- 최단거리 필요 없는 경우
- BFS/DFS 모두 가능

```python
# 1차원 상태
visited = [False] * (MAX + 1)

if not visited[nx]:
    visited[nx] = True
    q.append(nx)
    
# 2차원 격자
visited = [[False]*m for _ in range(n)]

if not visited[nx][ny]:
    visited[nx][ny] = True
    q.append((nx, ny))
```

---

#### B. `dist`배열로 방문처리
- 최단거리 문제
- BFS 기반 거리 계산

```python
dist = [-1] * (MAX + 1)
dist[start] = 0

if dist[nx] == -1:
    dist[nx] = dist[x] + 1
    q.append(nx)
```
